PowerShell et performances applicatives

Résumé

L'objectif de ce projet était de développer un robot effectuant des actions identiques à celles d'un utilisateur sur une application. Ce robot lançait une application, ouvrait différent menus et réalisait des actions graphiques qu'un utilisateur pourrait faire au quotidien, récupérait le temps de chargement des différents menus de l'application et envoyait ensuite ses données vers une base de données Oracle. Le logiciel de supervision ServiceNav de l'Institut de Cancérologie de Lorraine pouvait alors utiliser ces données pour produire des graphiques de performance de l'application. De cette manière, le support informatique pouvait anticiper les lenteurs avant que les utilisateurs n'appellent pour les signaler.

Compétences travaillées

  • Travail en autonomie
  • Organisation
  • Communication des résultats
  • Réalisation de documentation
  • Création de diagrammes UML
  • Progammation en PowerShell avec AutoIt(sous forme d'addon PowerShell)
  • Connexion de base de données à PowerShell
  • Connexion entre une solution de supervision et une base de données
